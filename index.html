ต้องการจัดการข้อมูลการจอง?

เข้าสู่ระบบผู้ดูแลทันที

สำหรับนักเรียน

  1. เลือกวิชาจากหน้าหลัก
  2. เลือกที่นั่งที่ว่าง (สีฟ้า)
  3. กรอกชื่อ-นามสกุลและห้องเรียน
  4. คลิกยืนยันการจอง
  5. บันทึกรหัสการจองที่ได้

สำหรับอาจารย์

  1. คลิกปุ่ม "ผู้ดูแลระบบ" ด้านบนขวา
  2. ล็อกอินด้วยรหัสผ่านที่ได้รับ
  3. ดูรายชื่อนักเรียนที่จอง
  4. ดาวน์โหลดรายชื่อเป็น Excel
  5. ตรวจสอบสถิติการจอง

สำหรับผู้ดูแลระบบ

  1. ล็อกอินด้วยบัญชี admin
  2. เพิ่ม/ลบ/แก้ไขวิชาเรียน
  3. จัดการข้อมูลการจองทั้งหมด
  4. ลบการจองที่ไม่ต้องการ
  5. ดูรายงานสถิติแบบเรียลไทม์

คำถามที่พบบ่อย

สามารถยกเลิกการจองได้หรือไม่?

กรุณาติดต่ออาจารย์ผู้สอนหรือผู้ดูแลระบบเพื่อยกเลิกการจอง

ต้องการเปลี่ยนที่นั่งทำอย่างไร?

ต้องยกเลิกการจองเดิมก่อน แล้วจองที่นั่งใหม่

พบปัญหาการใช้งานระบบ?

ติดต่อฝ่ายเทคโนโลยีสารสนเทศ หรือคลิก เข้าสู่ระบบผู้ดูแล เพื่อแจ้งปัญหา
